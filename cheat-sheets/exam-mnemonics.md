# ðŸ’¡ Exam Mnemonics & Memory Aids

Use these quick mnemonics to recall key concepts for the AWS Certified AI Practitioner Exam (AIF-C01).

## Responsible AI: Core Features (D4.1)

**F-A-V-B-I-S-R-I** (F-A-V-B-I is R-I)
* **F**airness
* **A**ccountability
* **V**eracity (Truthfulness)
* **B**ias
* **I**nclusivity
* **S**afety
* **R**obustness
* **I**nterpretability (Explainability)

## Evaluation Metrics (D3.4)

**R-O-U-G-E B-L-E-U**
* **ROUGE:** Best for **Summarization** tasks (Recall-Oriented).
* **BLEU:** Best for **Machine Translation** tasks (Bilingual Evaluation).
* **BERTScore:** Contextual similarity (often correlates best with human judgment).

## Prompt Engineering Techniques (D3.2)

**Z-S-F-C** (Zero-Shot, Few-Shot, Chain-of-Thought)
* **Z**ero-Shot: No examples in the prompt (simplest).
* **S**ingle-Shot: One example provided (part of Few-Shot).
* **F**ew-Shot: Several input/output examples provided (improves accuracy).
* **C**hain-of-Thought (CoT): Prompting the model to "think step-by-step" (improves reasoning).

## ML Learning Types (D1.1)

**S-U-R** (Supervised, Unsupervised, Reinforcement)
* **S**upervised: **Labeled** data (Classification, Regression).
* **U**nsupervised: **Unlabeled** data (Clustering, Dimensionality Reduction).
* **R**einforcement: **Action** based on rewards/penalties (Optimizing steps).
